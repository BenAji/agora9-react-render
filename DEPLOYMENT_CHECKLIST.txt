TROUBLESHOOTING QUICK REFERENCE
═══════════════════════════════════════════════════════════

Build Fails:
→ Check build logs in Render dashboard
→ Verify package.json has all dependencies
→ Clear build cache and redeploy

Can't Login:
→ Check Render URL is in Supabase redirect URLs
→ Verify using "anon" key (not service_role)
→ Check Supabase URL is correct

Environment Variables Not Working:
→ Verify prefixed with REACT_APP_
→ Check for typos (case-sensitive)
→ Redeploy after adding variables

Routes Return 404:
→ Verify render.yaml exists in root
→ Check publish directory is "build"
→ Redeploy from Render dashboard

Slow Performance:
→ Free tier has cold starts (15 min inactivity)
→ Upgrade to $7/month for instant response
→ Check bundle size optimization
